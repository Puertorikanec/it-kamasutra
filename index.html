.git Изучание

Изученные команды

  • git clone_repository - клонирование репозитория.
  • git status - состояние текущего локального репозитория
  • git add + название файла - добавление нового файла для отслеживания git'ом
  • git add . - добавить все новые файлы
  • git commit -am - добавление нового файла index.html на локальный репозиторий и дать имя комментарию.
  • git push - отправить в облачный репозиторий
  • git pull - получить изменения из облачного репозитория
