.git Изучание

Изученные команды

  • git clone_repository - клонирование репозитория.
  • git status - состояние текущего локального репозитория
  • git add + путь к файлу - добавление нового файла для отслеживания git'ом
  • git commit -am - добавление нового файла index.html на локальный репозиторий
  • git push - отправить в облачный репозиторий
  • git pull - получить изменения из облачного репозитория
